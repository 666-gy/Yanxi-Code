特性

边写边学，代码不言自明

八个核心能力，为编程学习者量身打造

边写边译

代码实时 AI 翻译解释，每写一行立即理解。三段式解读：做了什么、知识点、为什么这样写。

多语言支持

覆盖 Python、JavaScript、TypeScript、Java、C/C++、HTML、CSS 等主流编程语言。

AI 驱动

基于 DeepSeek 大语言模型，精准解读代码逻辑与设计意图，流式逐字输出。

Monaco 编辑器

VS Code 同款编辑器内核，语法高亮、行号、智能补全，专业编码体验。

Agent 助手

内置 AI Agent，支持读写、增删改查文件操作，像对话一样管理你的代码。

画布模式

Yan Board 画布空间：上传代码文件，AI 智能解析关键逻辑，一目了然。

工作区管理

文件夹即工作区，文件树浏览，多标签页切换，快捷键操作一气呵成。

深浅主题

深色/浅色主题自由切换，保护视力的同时适配不同编码习惯与工作环境。

更新公告

v1.1.0 更新内容

2026 年 6 月 30 日发布，多项修复与画布空间重做

重启后 API 配置不记忆

重启 Yanxi Code 后 API Key 配置丢失，须重新配置。已修复 API 密钥持久化存储逻辑，支持从多个 localStorage 键值双保险读取。

无法识别和打开 HTML 文件

工作区文件树无法识别 .html 文件类型，现已修复文件过滤器逻辑，支持 HTML 文件的打开与编辑。

画布空间重做 · Yan Board

全新 Yan Board 画布空间上线！写死工作区机制——现在只需点击"上传文件"按钮，选择代码文件，AI 自动解析分析。支持多文件、多 Tab 切换、最近打开历史，去掉不必要的复杂交互。

Markdown 文件无法预览

工作区中 .md 文件无法渲染预览，现已支持 Markdown 文件的即时预览渲染。

工作区修复

修复工作区文件夹打开、文件树同步、路径解析等多处稳定性问题，提升整体使用体验。

开发者从这里开始

参与贡献或自己动手构建，Node.js 18+ 即可

# 1. 克隆仓库
git clone https://github.com/666-gy/Yanxi-Code.git

# 2. 安装依赖并启动
npm install && npm run dev

# 3. 打包桌面应用
npm run electron:build
