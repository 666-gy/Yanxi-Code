01 / EDITOR

一个安静、完整，围绕真实文件工作的代码空间。

A quiet, complete coding space built around real files.

Yanxi Code 使用 Monaco Editor 提供多标签编辑、语法高亮、智能补全、Markdown 预览和未保存保护。文件树会实时响应工作区外部变更。

Yanxi Code uses Monaco Editor for tabs, highlighting, suggestions, Markdown preview, and unsaved-change protection. The file tree follows external workspace changes in real time.

02 / HANDOFF

编辑器与 Agent，不再是两套割裂的工作区。

The editor and the agent no longer live in separate workspaces.

从 Yanxi Code 唤起 Yan Agent 并传递当前项目，或由 Yan Agent 反向打开工作区。双端只交换本地路径与请求状态，不复制你的项目文件。

Launch Yan Agent with the active project, or let Yan Agent open a workspace in Yanxi Code. The two apps exchange only local paths and request state, never copies of the project.

03 / YAN TEACH

不只生成代码，也让代码变得可理解。

Go beyond code generation. Make code understandable.

选中代码或指定完整文件，Yan Teach 通过 DeepSeek 流式生成中文讲解，并把结果保存为工作区内可追踪的 Markdown。

Select code or a complete file. Yan Teach streams a Chinese explanation through DeepSeek and stores it as traceable Markdown in the workspace.
