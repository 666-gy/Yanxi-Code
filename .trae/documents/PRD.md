# Decipher — 边写边译 IDE 产品需求文档

## 1. 产品概述

Decipher 是一款面向初级开发者的学习型代码编辑器，核心特色是"边写边译"——开发者一边写代码，AI 一边实时翻译解释代码的含义、知识点和设计意图。通过 DeepSeek API 接入大语言模型，让编程学习从"查资料"变成"写中学"。

- **目标用户**：编程初学者、转行学习者、对新语言不熟悉的开发者
- **核心价值**：降低编程入门门槛，让代码不再是"天书"，每写一行都能立刻理解
- **差异化**：不是聊天式 AI 助手，而是嵌入编辑器的"实时翻译器"，右栏常驻解释面板

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 普通用户 | 无需注册，本地使用 | 使用全部编辑和翻译功能，配置 DeepSeek API Key |

### 2.2 功能模块

1. **工作区管理**：打开文件夹作为工作区，文件树浏览，多文件切换
2. **代码编辑器**：语法高亮、行号、多语言支持、基础编辑功能
3. **实时翻译面板（核心）**：右侧常驻面板，实时解释当前光标所在代码块
4. **深译模式**：选中代码后手动触发，获得更详细的逐行解释
5. **设置面板**：配置 DeepSeek API Key、模型选择、翻译延迟等
6. **状态栏**：显示当前语言、AI 连接状态、光标位置

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 主界面 | 顶部标题栏 | 工作区名称、AI 状态指示、设置按钮、菜单入口 |
| 主界面 | 左侧文件树 | 工作区内文件目录展示，点击切换文件，支持折叠 |
| 主界面 | 中央编辑器 | Monaco 代码编辑器，语法高亮，行号，光标跟随 |
| 主界面 | 右侧翻译面板 | AI 实时翻译解释，流式输出，分三段展示（做了什么 / 知识点 / 为什么这样写） |
| 主界面 | 底部状态栏 | 当前文件语言、光标行列位置、AI 连接状态指示 |
| 设置弹窗 | API 配置 | DeepSeek API Key 输入、模型选择、API 地址配置 |
| 设置弹窗 | 偏好设置 | 翻译触发延迟（debounce）、自动翻译开关、主题切换 |

## 3. 核心流程

### 主流程：边写边译

用户在编辑器中编写代码 → 输入暂停 300ms 后自动触发 → 提取光标所在的逻辑代码块 → 调用 DeepSeek API 流式生成解释 → 右侧翻译面板逐字渲染显示 → 用户继续写代码，循环往复

```mermaid
flowchart LR
    A["用户编写代码"] --> B["debounce 300ms"]
    B --> C["提取光标所在代码块"]
    C --> D["调用 DeepSeek SSE"]
    D --> E["流式渲染翻译面板"]
    E --> F["用户继续编写"]
    F --> A
```

### 深译模式流程

用户选中一段代码 → 右键菜单点击"深译" / 快捷键 → 直接提取选中内容 → DeepSeek 返回详细逐行解释 → 翻译面板展示更丰富内容

## 4. 用户界面设计

### 4.1 设计风格

**美学方向：赛博学究 (Cyber Scholar)**

深色编程环境 + 温暖学术感的翻译区，营造"科技图书馆"的氛围。

- **主色调**：深蓝靛青 `#0f172a`（编辑器背景）
- **强调色**：琥珀金 `#f59e0b`（AI 相关元素、标题高亮）
- **次要色**：青碧 `#06b6d4`（交互元素、状态指示）
- **翻译区背景**：深石板灰 `#1e293b`（略浅于主背景，区分层次）
- **文本色**：暖灰白 `#e2e8f0`（主文本）、琥珀金 `#fbbf24`（标题/重点）
- **代码字体**：JetBrains Mono（等宽，编程感）
- **翻译字体**：Source Serif Pro（衬线字体，学术感、可读性强）
- **UI 字体**：系统无衬线字体

**交互与动画**：
- AI 流式输出时的打字机效果 + 光标闪烁
- 翻译面板内容淡入滑入动画
- 状态指示的呼吸灯效果（AI 思考中）
- 文件切换的平滑过渡
- 悬停按钮的微光效果

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|---------|
| 主界面 | 顶部标题栏 | 瘦高条设计，左侧工作区名+琥珀点，中间logo，右侧设置按钮+AI状态灯 |
| 主界面 | 左侧文件树 | 可折叠侧栏，文件夹图标+文件名，悬停高亮，选中琥珀色底边 |
| 主界面 | 中央编辑器 | Monaco 深色主题，行号，当前行高亮，光标行有微光指示 |
| 主界面 | 右侧翻译面板 | 三段式卡片布局，每段有琥珀色小标题，Markdown 渲染，代码块深色背景 |
| 主界面 | 底部状态栏 | 极细条，左侧语言标签，中间光标位置，右侧 AI 状态（就绪/思考中/离线） |
| 设置弹窗 | 表单 | 毛玻璃背景，圆角卡片，输入框有焦点光晕，保存按钮琥珀色渐变 |

### 4.3 响应式

- **桌面优先**：IDE 类产品主要在桌面使用，最低宽度 1024px
- **侧栏可折叠**：左右侧栏均可折叠，给编辑器更大空间
- **面板宽度可调**：翻译面板宽度可拖拽调整

### 4.4 动效设计

- **首次加载**：各区域错落淡入（staggered reveal），从左到右依次出现
- **AI 思考中**：状态栏呼吸灯 + 翻译面板顶部三点跳动加载动画
- **流式输出**：逐字显示 + 末尾闪烁光标
- **翻译完成**：面板底部出现"✓ 翻译完成"微提示，1秒后消失
- **切换文件**：编辑器内容交叉淡入淡出
